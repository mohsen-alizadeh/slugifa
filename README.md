### Slugifa

Simple gem to make slug which supports Persian (Farsi) characters.
Follow following easy steps.

1. gem install slugifa

2. use it

#### examples

``` Ruby
'Mohsen Alizadeh presents'.slugifa
# => "mohsen-alizadeh-presents"

'برگزاری‌ آنلاین مسابقات'.slugifa
 # => "برگزاری-آنلاین-مسابقات"

```
